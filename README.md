# Cek Masa Aktif Kartu Perdana XL/Axis

## Usage
1. Menginstall software/runtime [Bun](https://bun.sh)
2. Ketik `bun i` pada directory software ini
3. Isi file config.txt
4. Jalankan `bun run src/index.ts config.txt`
5. Enjoy!
